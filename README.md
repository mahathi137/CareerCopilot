## 🎯 Features

CareerCopilot is a comprehensive placement preparation platform designed for students to master DSA, aptitude, and interview skills.

### Core Modules

- **💻 DSA Practice** - 40+ curated problems from Easy to Hard difficulty
- **🧮 Aptitude Hub** - 30+ quantitative, logical, and verbal questions
- **🎤 Interview Prep** - 30+ HR, technical, and behavioral questions with tips
- **📄 Resume Master** - Interactive checklist to build a perfect resume
- **🏢 Company Prep** - Detailed profiles for TCS, Infosys, Wipro, and more
- **🗓️ Study Planner** - Organize your preparation with a structured plan
- **🏅 Achievements** - Earn badges and track your progress
- **📊 Analytics** - Detailed insights into your preparation progress
- **🤖 AI Copilot** - Intelligent chatbot assistant available 24/7

### Key Features

- **Local Storage** - All progress saved in browser, no backend required
- **Dark/Light Theme** - Toggle between themes
- **Responsive Design** - Works seamlessly on mobile devices
- **XP System** - Earn XP for completing tasks
- **Streak Tracking** - Maintain daily study streaks
- **Placement Readiness Score** - Comprehensive score based on all modules
- **GitHub Pages Compatible** - Deploy directly to GitHub Pages

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required - runs entirely in the browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/careercopilot.git
cd careercopilot
```

2. Open `index.html` in your browser:
```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

### Deploy to GitHub Pages

1. Push the repository to GitHub
2. Go to repository Settings → Pages
3. Select `main` branch as source
4. Your site will be live at `https://yourusername.github.io/careercopilot/`

## 🎨 Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables
- **Vanilla JavaScript** - No frameworks, pure JS
- **Chart.js** - Data visualization
- **Local Storage API** - Client-side data persistence

## 📊 Data Persistence

All user data is stored in the browser's Local Storage:
- Completed problems
- Aptitude scores
- Interview bookmarks
- Resume checklist
- Study planner settings
- Profile information
- Achievements and XP

### Export/Import

Users can export their progress as JSON and import it on another device using Settings → Export/Import.

## 🎯 Placement Readiness Score
